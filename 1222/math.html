01 자바스크립트 내장객체_수학(math)객체

수학객체 : 다른 내장객체와 다르게 new키워드를 사용하지 않는 정적인 객체이다.

객체(object) : 메서드 / 속성으로 나뉜다.

    ~~~~ 객 체 ~~~~

  • 메서드 (동사) : 기능적인 동작을 말한다, write();
  • 속성 (명사) : 서식등을 말한다, style.color='서식내용'


수학객체의 속성들

  1. e : 상수

  2. PI : 원주율

  3. SQRT2

  4. SQRT1_2

  5. LN2

  6. LN10

  7. LOG2E

  8. LOG10E

수학객체의 매서드 종류

  1. max(숫자1, 숫자2, 숫자3 ...) : 숫자들 중에서 최대값을 반환

  2. min(숫자1, 숫자2, 숫자3 ...) : 숫자들 중에서 최대소값을 반환

  3. round(숫자) : 소수점 첫째 자리를 반올림

  4. ceil(숫자) : 소수점 자리를 무조건 올림

  5. floor(숫자) : 소수점 자리를 무조건 절삭

  6. abs(숫자) : 수의 절대값을 반환

  7. random(숫자) : 0과 1사이의 난수 표시
