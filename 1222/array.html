02 자바스크립트 내장객체_배열(array)객체

배열(array)객체 이란?
  1. 연관된 데이터를 하나처럼 관리하고 싶을 때 사용
  2. 복수의 데이터를 제어할 수 있는 다양한 함수 제공
  3. 배열의 함수를 이용한 데이터의 편리한 조작이 가능


1. 배열 객체 선언하기

방법 (1)
let array = new Array(data1, data2, data3 data4 ... dataN);


방법 (2)
let array = [data1, data2, data3 data4 ... dataN];


예시 (1)
let fruits = new Array('수박', '참외', '귤', '감', '사과', '포도', '복숭아' ...)

예시 (2)
let fruits = ['수박', '참외', '귤', '감', '사과', '포도', '복숭아' ...]



2. 배열 객체 호출하기

배열객체의 데이터를 호출하고자 할 때는 인덱스 번호를 사용하는데 번호는 0 부터 시작한다.



실습예제. 중화요리 음식 이미지를 사용하여 요일별 음식 추천하기

날짜 객체를 사용하여 현재 요일 번호를 구해서 배열번호에 맞는 이미지 파일명을 찾아 img태그에 대입해서 음식 사진이 요일별로 출력되게 한다.
