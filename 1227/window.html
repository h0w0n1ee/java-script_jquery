윈도우(브라우저)객체 : 사용자가 사용하는 웹브라우저의 창에서 다양한 동작이나 속성을 설정하고자 할 때 사용하는 객체



window 객체

window객체 : 브라우저 객체중 최상위 객체로서 현재 실행되고 있는 브라우저창을 말함



window객체의 속성

  • document

  • frameElement

  • innerHeight

  • innerWidth

  • location

  • name

  • screenX

  • screenY

  • screenX, screenY

  • sessionStorge


window객체의 메서드

  • alert( ) : 윈도우창에서 경고창(메세지)를 띄울 때 사용

  • blur( ) : 현재 창에서 포커스를 제거

  • close( ) : 현재 창을 닫기

  • confirm( ) : 확인/취소 버튼이 있는 확인창을 표시

  • focus( ) : 현재 창에 포커스(초점)를 부여

  • moveBy( ) : 현재 창을 지정한 크기만큼 이동

  • moveTo( ) : 현재 창을 지정한 좌표로 이동

  • open( ) : 새창 열기

  • prompt( ) : 사용자가 입력 가능한 프롬포트 창을 열기

  • resizeTo( ) : 동적으로 브라우저 창의 크기를 조절

  • scroll( ) : 문서의 특정 위치로 스크롤하기

  • sizeToContent( ) : 내용에 맞게 창 크기 조절

  • stop( ) : 로딩 중지












모달윈도(modal window)
사용자 인터페이(UI)디자인 개념에서 자식 윈도에서 부모 윈도우로 돌아가기전에 사용자의 상호동작을 요구하는 창을 말한다. 모달 윈도우는 부모창을 배경으로 어둡게 하고 자식요소가 화면의 가운데 중앙에 배치하도록 하며, 오른쪽 상단에 '닫기'버튼을 만들어서 닫을 수 있도록 한다.

모달윈도 띄우기 4가지 연습하기

  1. 배경없이 광고만 띄우기
  2. 배경 어둡게 투명도 적용하여 광고 띄우기
  3. 버튼 클릭하면 모달윈도 띄우기
  4. 사용자가 페이지 접속하면 모달윈도 띄우기
  5. 닫기 버튼 클릭시 모달윈도 숨기기
