01데이터의 유형(data type)

자바스크립트에서 사용하는 데이터의 유형에 대해 알아본다.

  • 문자 데이터형 (string)
  • 숫자 데이터형 (number)
  • 부울형 (boolean) : true(확인), false(취소)
  • 찾을 수 없는 데이터형 (undefined)
  • 값이 없는 데이터 (null)
  • 배열형 데이터 (array)
  • 함수형 데이터 (function)
  • 객체 (object) : document, window, alert, confirm, prompt ...

typeof 연산자 : 사용자가 입력한 데이터가 어떤 종류의 형식인지를 알려주는 연산자
변수 앞에 Number, String등을 작성하면 실제 데이터 형식을 알 수 있다.

es6문법에서 새롭게 추가된 기호 >> 템플릿 리터럴 (`)
연산자를 사용하지 않아도 간단한 방법으로 새로운 문자열을 삽입할 수 있는 기능을 제공한다. 이를 문자열 인터폴레이션 (String Interpolation)이라 한다.
